# Gonki
# ◧ GONKI — The Website Roaster

> *“The web is fragile. Let’s find the cracks.”*

Gonki is a playful website roaster that scans a target URL and delivers sarcastic, witty, and brutally honest feedback about design, UX, and frontend decisions.

Built inside a retro desktop-inspired interface, Gonki blends humor with analysis to turn website critique into entertainment.



## ✨ Overview

Gonki simulates a fake operating system called **Gonki OS**, where users run a tool called **Roaster.exe**.

Users paste a website URL → Gonki performs a “system scan” → then generates:

- 🔥 Roasts are aren't ai generated they are manually fed
- 📊 Integrity Score (out of 100)
- 🧠 Personality-driven feedback

The experience is designed to feel like running an old-school desktop utility — but with modern sarcasm.

## ⚡ Features

### ◧ Retro Desktop Environment
- OS interface
- System bar & desktop icons
- Draggable window UI

### 🤖 Roast Engine
- Generates sarcastic website feedback which are already in the database
- Frontend & UX themed humor
- A fun Project

### 📊 Integrity Score
- Final score rating for the scanned website
- Presented like a system diagnostic

### 💻 Terminal-style Analysis
- Scanner logs
- Progress bar simulation
- System-style interaction flow

### 🗑️ Easter Eggs
- Recycle Bin roast messages
- Passive-aggressive system status
- Fake system properties

---

## 🧩 How It Works

1. User enters target URL
2. Gonki starts "System Analysis"
3. Simulated scan + logs appear
4. Roast results are generated
5. Integrity score is displayed
